
How to start a simple HTTP server?
----------------------------------

`flask` is a lightweight web server.
You can create APIs and simple web pages with very little code.

Install it with:

.. code::

    pip install flask

Here is an example server (store the code in `myapp.py`).
Also place the file :download:`templates/main.html` in a folder `templates/`
:

.. literalinclude:: run_flask_server.py

Then run the server with:

.. code::

    python myapp.py
